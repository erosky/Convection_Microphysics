SPEC Learjet 35 Instrument Data
https://data.eol.ucar.edu/dataset/605.020

Summary
State and Cloud microphysics data collected on board the SPEC Learjet 35
during the Secondary Production of Ice in Cumulus Experiment (SPICULE)
campaign from 29 May through 25 June 2021. This data set is in ICARTT format.
Please see the ReadMe file and the header portion of the data files for
details on instruments, parameters, quality assurance, quality control,
contact information and data set comments.
